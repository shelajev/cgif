Important info about the service

First of all, the visual assets like the board image and the pieces were taken from the Lichess.org. They are available from their Github repository under MIT license.

Thank you, Lichess!

This service was created because I need a test Play Framework application. So the quality is not really the swiss watch like.

It's deployed to Heroku, because they have out-of-the-box support for Play apps. It's powered by their free dyno, so it has really limited resources and it might fall down any moment. Also, the service doesn't use any persistence options, it all sits in the memory cache (which should be pretty small for the free dyno), so if you share a link to a gif it might not work after that. In fact, it won't work for certain after some time, just like Snapchat, but for other reasons.

Save those images locally, if you like them.

Oh, this also uses Bootstrap to look less hideous, which includes free use of Glyphicons, that otherwise are not.
